# Reward App Frontend (Bootstrap)

This folder contains the initial React + Vite frontend scaffold for the reward app.

## Included
- React 18 + Vite setup
- Client-side routing (`react-router-dom`)
- Starter routes:
  - Offer Wall (`/`)
  - Wallet (`/wallet`)
  - Profile (`/profile`)
  - Conversion History (`/history`)
- Vite proxy to backend API on `http://localhost:3000`

## Run locally
```bash
cd frontend
npm install
npm run dev
```

Backend should run separately from repository root:
```bash
npm install
npm run dev
```
